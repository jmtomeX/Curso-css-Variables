Valores de sustitución (fallback) de propiedades personalizadas

Utilizando var() puede definir múltiples valores de sustitución (fallback) cuando la variable dada no está definida aún, esto puede ser útil cuando se trabaja con Custom Elements y Shadow DOM.

El primer argumento a la función es el nombre de la propiedad personalizada que se va a sustituir. El segundo argumento a la función, si se proporciona, es un valor de reserva, que se utiliza como valor de sustitución cuando la propiedad personalizada referenciada no es válida
