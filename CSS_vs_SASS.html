Sass vs Css

* Sass se pueden declarar sin selectot
* Las variables tienen scope de bloque
* Operaciones aritméticas ya compiladas
* Puede ser interpoladas. Se pueden construir otra expresion un selector, una propiedad o una función. * Se le pueden pasar variables a las mediaQuerys.
* No se pueden redefinir.
* Las variables no existen depués de la compilación.

* CSS necesitan un selector
* Las variables tienen scope de DOM, es decir del padre a todos sus descendientes
* Operaciones aritméticas calc(), las compila el navegador
* No se pueden usar funciones de color para manipular colores.
* No pueden ser interpoladas.
* No se le pueden pasar variables a las mediaQuerys, el navegador no las entiende.
* Se pueden redefinir.
* Las variables existen en el DOM.
